# PACTA - Contract Management System

## Installation

### Linux (Ubuntu/Debian)

```bash
tar -xzf pacta-v*-linux.tar.gz
cd pacta-v*/linux
sudo bash install.sh
```

Access at: http://127.0.0.1:3000

### Windows

1. Extract `pacta-v*-windows.zip`
2. Run `install.bat` as Administrator
3. Access at: http://127.0.0.1:3000

## First Setup

1. Open http://127.0.0.1:3000 in your browser
2. Complete the setup wizard to create admin account
3. Login with your admin credentials

## LAN Access

To allow other computers on the network to access PACTA:

**Linux:** `sudo ufw allow 3000/tcp`
**Windows:** Add inbound rule for port 3000 in Windows Defender Firewall
